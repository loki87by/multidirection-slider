# multidirection-rock-slider

### Музыкальный слайдер с возможностью прокрутки слайдов как горизонтально, так и вертикально кликом или свайпом слайда.

## Технологии используемые в проекте: 
* Медиаэлементы 
* Относительные величины 
* Модификаторы 
* Псевдоэлементы 
* Семантические теги 
* Адаптивная вёрстка 
* Flex-контейнеры 
* и прочее
